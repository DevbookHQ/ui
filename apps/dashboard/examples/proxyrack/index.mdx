## Premium Residential
### Geo-Target a Country

This endpoint provides access to unique monthly residential IP addresses and has a high success rate.

HTTPS and SOCKS protocols are enabled on this service.

* You can geo-target a country by adjusting the username, like this `const username = 'yourUsername-country-US'`
* You can then check to make sure this works properly by making a request to the 'stats' API endpoint. Under `requestParams` you will see this reflected as `country`.

{`  async function authWithEmail() {
    setIsLoading(true)

    const email = emailRef.current?.value
    const password = passwordRef.current?.value

    if (!email) {
      setErrMessage('Email must not be empty')
      emailRef.current?.focus()
      setIsLoading(false)
      return
    }

    if (!password) {
      passwordRef.current?.focus()
      setErrMessage('Password must not be empty')
      setIsLoading(false)
      return
    }

    const { error, data: { user } } =
      authType === AuthFormType.SignUp
        ? await supabaseClient.auth.signUp({
          email,
          password,
        })
        : await supabaseClient.auth.signInWithPassword({
          email,
          password,
        })

    if (error) {
      emailRef.current?.focus()
      setErrMessage(error.message)
      console.error(error.message)
    } else {
      setErrMessage('')
    }

    posthog.identify(user?.email, {
      email: user?.email,
    })

    router.replace('/projects')
    setIsLoading(false)
  }
`}
