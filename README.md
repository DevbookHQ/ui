# Devbook UI
Devbook UI is a React UI library.

## Installation
```sh
npm install @devbookhq/ui
```

## Usage
Available components
- Editor
- Output
- Iframe

Example of importing a component.
```tsx
import { Editor } from '@devbookhq/ui'

return <Editor content="console.log()"/>
```
